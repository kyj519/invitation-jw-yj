# invitation-jw-yj

결혼식 초대장 웹페이지

## GitHub Pages 배포

이 프로젝트는 GitHub Pages를 통해 자동으로 배포됩니다.

### 배포 설정

1. 저장소 Settings > Pages로 이동
2. Source를 "GitHub Actions"로 설정
3. main 또는 master 브랜치에 푸시하면 자동으로 배포됩니다

### 웹사이트 접속

배포 후 다음 주소에서 확인할 수 있습니다:
- `https://kyj519.github.io/invitation-jw-yj/`

## 로컬 개발

1. 저장소를 클론합니다
2. `index.html` 파일을 브라우저에서 엽니다

## 파일 구조

- `index.html` - 메인 HTML 파일
- `style.css` - 스타일시트
- `.github/workflows/deploy.yml` - GitHub Actions 배포 워크플로우
